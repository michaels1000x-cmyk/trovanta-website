# Trovanta Ltd — Company Website

A self-contained static website. No build step, no dependencies.

## Files

- `index.html` — all page content
- `styles.css` — design system and layout
- `script.js` — mobile menu, scroll animations, footer year

## Preview locally

```sh
python3 -m http.server 4173 --directory .
```

Then open http://localhost:4173

## Deploy

Upload the three files to any static host — Netlify, Vercel, GitHub Pages,
Cloudflare Pages, or a standard cPanel/shared host. No configuration needed.

## Before going live

- Optionally swap the text-based "T" logo mark for a real logo file.
